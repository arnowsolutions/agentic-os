# Example Data Files

This directory contains example/sample data for reference purposes.
These are NOT used in production runtime - they're for testing and documentation.

## Files

- `goals_example.json` - Sample goal structure
- `voice_messages_example.json` - Sample voice message structure
- `crm_contacts_example.json` - Sample contact structure (if needed)

## Usage

Copy these examples when creating new records in the live system.
